# nodeBasic

## Command
### nodemon 実行
`npm run dev`

## Settings
- yarn install
`npm i yarn`  
- npm init
`npm init`  
- express install
`npm i express`  
- nodemon install(保存後に自動実行するライブラリ)
`npm i nodemon`  
- mongoose install
`npm i mongoose`  
- dotenv install
`npm i dotenv`
